# dbpedia-stats
Stats generator and widgets for DBpedia Databus collections

## Deployment

```
git clone https://github.com/holycrab13/dbpedia-stats.git
cd dbpedia-stats/baker
docker build -t stats-baker .
cd ../web-app
docker-compose up
```

Open web app at 

```
localhost:5000
```
